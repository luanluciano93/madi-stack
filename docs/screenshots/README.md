# Screenshots

Capturas usadas no README principal. Ordem e nomes esperados:

| Arquivo               | Conteúdo sugerido                                                        |
| --------------------- | ------------------------------------------------------------------------ |
| `geral.png`           | Aba **Geral** com os 3 serviços rodando (LED verde, PID, porta)          |
| `sites.png`           | Aba **Sites** com ao menos 1 vhost ativo + botão Root Dir + HTTPS toggle |
| `atualizacoes.png`    | Aba **Atualizações** após um "Verificar tudo"                            |
| `firewall.png`        | Aba **Firewall** com as 3 regras presentes                               |
| `configuracoes.png`   | Aba **Configurações** com portas, idioma e tema                          |
| `logs.png`            | Aba de um serviço (Nginx/MariaDB) com LogViewer cheio                    |

Dimensões: 1280×800 (ou largura maior se quiser alta resolução — README usa
`width="720"` para exibição). PNG ou WebP, idealmente < 400 KB cada.

Instruções rápidas:

1. Abra o `MadiStack.exe` em `target/release/`.
2. Ajuste pra tema escuro (ou o que preferir) e PT-BR.
3. Use **Win+Shift+S** (ferramenta de captura) e salve com o nome da tabela.
4. Coloque aqui em `docs/screenshots/` e commite junto com o release.
